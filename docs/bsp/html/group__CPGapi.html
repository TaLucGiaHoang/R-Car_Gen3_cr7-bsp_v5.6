uint32_t  ResetStat 
)

Set a state of the CPG reset for specified module.

Function R_PMA_CPG_SetResetStat

Parameters
[in]RegIdx- SRCR/SRSTCLR register index, range R_CPG_REGIDX_0 - R_CPG_REGIDX_11. See r_cpg_RegIdx_t
[in]BitPos- SRCR/SRSTCLR register bit number, range R_CPG_BIT0 - R_CPG_BIT31. See r_cpg_BitPos_t
[in]ResetStat- 1: Set SRCR target bit to 1. 0: Set the SRSTCLR target bit to 1 and wait for the SRCR bit to become 0. It is necessary to wait for RCLK 1 cycle on the user side.
Return values
Seer_cpg_Error_t
Note
In this API, it is possible to specify the Bit of the Module described in Table 8A.27 to Table 8A.38 of HWM. In addition, even if you specify a Bit that does not have Module described in the table, no error will occur.
