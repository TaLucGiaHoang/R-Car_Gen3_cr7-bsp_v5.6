r_wm_Error_t R_WM_DevDeinit ( r_wm_Dev_t  WmUnit)

De-initialise a Window Manager Device.

This function will close the communication channel with the CIO WM Device and the disable the corresponding DU, VSPD and DOC if they are not used by other application. DU and VSPD do not have DeInitialize API, therefore the corresponded registers will not clear. (Only disable DU and remove VSPD interrupt setting)

Parameters
[in]WmUnit- WM Device to be initialised. Correspong to the DU unit. Range R_WM_DEV_VOUT0 - R_WM_DEV_VOUT3. See r_wm_Dev_t
Return values
WMerror r_wm_Error_t
