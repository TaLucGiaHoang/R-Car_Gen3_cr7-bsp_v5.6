◆ R_WM_SurfacePropertySet()

r_wm_Error_t R_WM_SurfacePropertySet ( r_wm_Dev_t  WmUnit,
r_wm_SurfaceProp_t  Prop,
r_wm_Surface_t *  Surf 
)

Set the Surface property.

This function saves the request in the ringbuffer. Only when the function a screen flush is requested will the changes be sent to the driver via the OSAL communication channel. The property must be one of the define one, see r_wm_SurfaceProp_t.

  • R_WM_SURF_PROP_POS : The values of PosX, PosY, PosZ in Surf are used.
  • R_WM_SURF_PROP_SIZE : The values of Width, Height, StrideY, StrideC in Surf are used.
  • R_WM_SURF_PROP_COLORFMT : The values of Fmt in Surf are used.
  • R_WM_SURF_PROP_BUFFER : The values of FBuf, FBufC0, FBufC1 in Surf are used. : In case that a data of ARGB8888 format is set to this member variables, : Transparent is enable by setting Alpha component.
  • R_WM_SURF_PROP_DOC : The values of Doc in Surf are used. r_wm_Doc_t.
Parameters
[in]WmUnit- WM Device, range R_WM_DEV_VOUT0 - R_WM_DEV_VOUT3. r_wm_Dev_t
[in]Prop- Surface property to be set. r_wm_SurfaceProp_t
[in]Surf- pointer to a structure containting all the surface information. r_wm_Surface_t
Return values
WMerror r_wm_Error_t
