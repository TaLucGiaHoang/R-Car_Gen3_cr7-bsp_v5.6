This function de-register a channel to the given WM device If there are still open channels, it returns 1 through Par1. If Par1=1 is returned the WM device shall be de-initialised.

Parameters
[in]Addr- Reference to the WM device
[out]Par1- 0 if the WM device has no connection, else 1.
[in]Par2- not used
[in]Par3- not used
Return values
0if successful; -1 if failed
