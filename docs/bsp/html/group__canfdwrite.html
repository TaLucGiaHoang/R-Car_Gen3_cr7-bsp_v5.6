void *  Par1,
void *  Par2,
void *  Par3 
)

Write to the message buffer assigned to the given instance and initiate transmission.

R_CIO_CANFD_PRV_Write runs, and if the return value is OK, the channel interrupt occurs asynchronously. If run R_CIO_CANFD_PRV_Close before receive the channel interrupt, the transmission will not be completed.

Parameters
[in]Addr- Channel number of the given CANFD instance
[in]Par1- Buffer address (source)
[in]Par2- length of data to write. If 0 is specified, it is not an error. Also, the write data is returned as 0.
[in]Par3- optional parameter (unused)
Returns
>= 0 - length of write (handled) data
< 0 - NG
