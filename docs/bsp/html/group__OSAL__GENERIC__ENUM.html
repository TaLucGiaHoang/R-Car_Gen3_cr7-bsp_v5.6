◆ e_osal_mmngr_allocator_mode

enum e_osal_mmngr_allocator_mode

Common Memory Manager allocation mode values.

Enumerator
OSAL_MMNGR_ALLOC_MODE_INVALID 

Invalid/error value

OSAL_MMNGR_ALLOC_MODE_STACK 

Implementations may always fall to protection enabled mode OSAL_MMNGR_ALLOC_MODE_STACK_PROT

OSAL_MMNGR_ALLOC_MODE_STACK_PROT 

Same as stack mode but with underflow and overflow detection

OSAL_MMNGR_ALLOC_MODE_FREE_LIST 

FreeList(BestFit) Allocation mode. Implementations may always fall to protection enabled mode OSAL_MMNGR_ALLOC_MODE_FREE_LIST_PROT

OSAL_MMNGR_ALLOC_MODE_FREE_LIST_PROT 

Same as free list mode but with underflow and overflow detection

OSAL_MMNGR_ALLOC_MODE_UNSPECIFIED 

If Specific Memory Managers implement other allocation modes than OSAL API, they can set this value when R_OSAL_MmngrGetConfig() is called. This value is not expected as input configuration for R_OSAL_MmngrOpen()

OSAL_MMNGR_ALLOC_MODE_UNSPECIFIED_PROT 

If Specific Memory Managers implement other allocation modes than OSAL API with underflow/overflow detection, they can set this value when R_OSAL_MmngrGetConfig() is called. This value is not expected as input configuration for R_OSAL_MmngrOpen()

◆ e_osal_mmngr_event

enum e_osal_mmngr_event

Memory Manager Event enumeration values. Used to identify certain events that can be issued by an Memory Manager instance.

See also
R_OSAL_MmngrRegisterMonitor() for more details.
Enumerator
OSAL_MMNGR_EVENT_INVALID 

Invalid value

OSAL_MMNGR_EVENT_ALLOCATE 

Event issues after allocation completes

OSAL_MMNGR_EVENT_DEALLOCATE 

Event occurs before deletion of the buffer. Handle of the buffer is still valid!

OSAL_MMNGR_EVENT_OVERFLOW_DETECTED 

Event occurs if an buffer overflow is detected. Handle of the buffer is still valid.

OSAL_MMNGR_EVENT_UNDERFLOW_DETECTED 

Event occurs if an buffer underflow is detected. Handle of the buffer is still valid.

OSAL_MMNGR_EVENT_ERROR 

Reserved for the future update. Some error event is detected.
