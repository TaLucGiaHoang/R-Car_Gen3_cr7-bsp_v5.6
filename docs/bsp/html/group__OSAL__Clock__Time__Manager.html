e_osal_return_t R_OSAL_ClockTimeCalculateTimeDifference ( const st_osal_time_t *const  p_time2,
const st_osal_time_t *const  p_time1,
osal_nano_sec_t *const  p_time_difference 
)

Acquire the difference between two times.

Parameters
[in]p_time2The time acquired at R_OSAL_ClockTimeGetTimeStamp().
[in]p_time1The time acquired at R_OSAL_ClockTimeGetTimeStamp().
[in,out]p_time_differenceTo set the address of osal_nano_sec_t. The OSAL sets the difference between time2 and time1 with a signed value.
Returns
e_osal_return_t
Return values
OSAL_RETURN_OKAll arguments are valid, and the difference between two times was acquired successfully.
OSAL_RETURN_PARInvalid argument.
  • p_time_difference is NULL.
  • p_time1 or p_time2 is NULL.
  • p_time1->tv_sec or p_time2->tv_sec is out of range. please see also st_osal_time_t.
  • p_time1->tv_nsec or p_time2->tv_nsec is out of range. please see also st_osal_time_t.
OSAL_RETURN_FAILAn error occurred in OS or related API.
OSAL_RETURN_STATENot initialized. (Before calling R_OSAL_Initialize().)
Note
The result of p_time_difference is not guaranteed if clock type has different between time1 and time2. Regarding to time2, please set the value that after time of time1.
