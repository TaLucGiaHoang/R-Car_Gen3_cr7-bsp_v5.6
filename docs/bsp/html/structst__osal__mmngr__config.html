Detailed Description

configuration of Memory Manager.

(member variables of this structure)

  • memory_limit: Value range [4KB – OSAL_MEMORY_MANAGER_CFG_UNLIMITED-1] [Safety Criteria]
    If OSAL_MEMORY_MANAGER_CFG_UNLIMITED is specified in R_OSAL_MmngrOpen(), the maximum memory size area is allocated.
    The value acquired by R_OSAL_MmngrGetOsalMaxConfig() may be lower than the actual available memory and shall not be larger than the available memory.
  • max_allowed_allocations: Value range [1 – OSAL_MEMORY_MANAGER_CFG_UNLIMITED-1] [Safety Criteria]
    Implementations may use this value to allocate memory required for operation overhead.
    If OSAL_MEMORY_MANAGER_CFG_UNLIMITED is specified in R_OSAL_MmngrOpen(), this function does not allocate bookkeeping area.
  • max_registered_monitors_cbs: Value range [0 – OSAL_MEMORY_MANAGER_CFG_UNLIMITED-1] [Safety Criteria]
    Implementation may use this value to allocate memory required for operation overhead.
    If OSAL_MEMORY_MANAGER_CFG_UNLIMITED is specified in R_OSAL_MmngrOpen(), this function does not allocate an bookkeeping area for the monitor function.
