Detailed Description

Methods used by the CIO Server to initiate or finish communicatrion with the CAN-FD module.

Functions which a the CIO Server will call in order to initiate a connection between a user applcation and the CAN-FD macro.

CANFD internal Close function.

Stops one CANFD channel operation.

When R_CIO_CANFD_PRV_Close is called, the processing will be interrupted if there is a processing request immediately before.
