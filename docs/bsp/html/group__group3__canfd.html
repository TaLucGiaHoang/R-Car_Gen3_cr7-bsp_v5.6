#define R_CIO_CANFD_PRV_FOR_EACH_SET_BIT (   bit,
  mask,
  max 
)
Value:
for (bit = 0; bit < max; bit++) \
if ( ((0x1 << bit) & mask) != 0)

A conditional for-loop.

"Walk" the given mask value bit by bit. The operation inside the loop is only executed if the corresponding mask bit is set

Parameters
[in]bit- loop counter variable, actual bit number
[in]mask- mask value
[in]max- For loop limit (total number of bits)
Returns
void
